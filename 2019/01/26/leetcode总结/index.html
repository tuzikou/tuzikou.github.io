58. Length of Last Word

描述

思路一
用一个变量对单词从前往后进行计数，当前字符为空格时，重新计数，直到最后一个单词
注意在计数前，应当使用函数strip()移除字符串末尾可能出现的空格。

思路二
采用内置函数
Python rstrip() 删除 string 字符串末尾的指定字符（默认为空格）
Python split() 通过指定分隔符对字符串进行切片，如果参数 num 有指定值，则分隔 num+1 个子字符串，返回列表
